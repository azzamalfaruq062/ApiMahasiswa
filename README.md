
## Langkah Membuat
Alat yang digunakan :
1. Php versi 8.1
2. Laravel versi 8.3
3. MongoDB versi 4.2

Langkah-langkah :
1.Download MongoDB, kemudian install driver mongodb pada windous, dan edit file php.ini.
2.install laravel baru

```bash
composer create-project --prefer-dist laravel/laravel="^8.0"
```
```bash
$ composer require jenssegers/mongodb
```
